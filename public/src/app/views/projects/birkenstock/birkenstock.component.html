Birkenstock

2016 | Freelance Project

< All Projects

In 2016, I decide with the help of one of my friend to start a freelancer association in order to start gaining experience with client interaction. Few months later via this association we started working with a Birkenstock retailer from the iconic Bondi Beach. This client did not have any digital presence except social media. With the client we defined the needs and objectives for this website to reach.

In order to keep the trust of the user the choice was made to keep the graphic chart of the Birkenstock shoe brand as a base. But in order to fit the particular location, Bondi Beach, we integrated with some elements of beach, sun & ocean. The real challenge was for us to find the right balance between these two different universes

Code igniter was chosen as base, because of itâ€™s flexibility and efficiency. On top of that javascript was used to bring dynamism and interaction, the shoe filter is a good example of this interaction as it updates automatically the result when new filters are added. Some APIs have also been connected such as Mailchimp to help retaining the user alongside their visits. A big part of this website is actually not visible by the general user, in order to keep track of the shoes, collections and blog post a fully personalised backend has been build letting the employee update and manager the website by themselves.
